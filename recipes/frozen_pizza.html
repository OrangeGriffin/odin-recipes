Frozen Pizza

Description

You are in for a treat with this option. You like pizza? Don't want to leave the house? Read on for more details!

Ingredients

  • Pizza, frozen
  • Oven
  • Timer

Directions

      Preheat oven to indicated temperature
      Remove protective layer from pizza
      Place in oven
      Set timer to indicated baking time
      Wait
      Remove from oven when timer dings
      Slice and enjoy
