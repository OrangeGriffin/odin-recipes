Strawberry Rhubarb Crisp

Description

This should be where some fancy text is to describe the above dish. But let's be honest, I don't know how to describe a strawberry rhubarb crisp.

It looks like there is a lot involved. Probably need some strawberry, and some rhubarb, and some crisp.

Ingredients

  • 2 Pounds Rhubarb Stalks, sliced 1/2 inch thick
  • 1 1/4 cups sugar
  • 3 tablespoons cornstarch

Directions

  1. Combine ingredients
  2. Preheat oven to 450 degrees
  3. Bake for 45 minutes
